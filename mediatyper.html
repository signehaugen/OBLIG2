Media Typer

Media typer er forskjellige plattformer hvor man kan se sitt digitale innhold. Noen digitale plattformer kan være Print, Handheld og Screen. Print er det innholdet som kommer på arket når du printer ut. Om man lager ett eget stil ark i css for print vil det få et nytt utseendet på papiret. For eksempel om du bare vil at teksten på siden din skal printes ut må du gjøre om dette i css filen din. Handheld media type er en css-spesifikasjon for håndholdte enheter. Dette mediet er nesten dødt på grunn av at detresponsive designet tar over å gjør om størrelsen uansett størrelse. Screen er mediet med det «vanlige» formatet. Alt vi ser nettsider/applikasjoner på i våre dager er skjerm/screen. (Stasjonær pc, laptop, nettbrett og smarttelefon). Dette er nyttig for å skille stil ark tilpasset ulike medietyper (skjerm vs print). Disse css egenskapene er svært relevante for å kunne få en fin, oversiktlig og strukturert siden, uansett hvilke media typer du bruker.
