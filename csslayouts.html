CSS Layouts

En CSS layout er måten elementene i designet er satt opp. Dette vil gjøre så du får en bedre struktur på siden din. Man har fire forskjellige layouts som man kan bruke. Det er CSS Float, CSS Grid, CSS Flexbox og tabellutforming som ble brukt for flere år siden.

CSS Grid gir oss en metode for å strukturere innholdet ved hjelp av et rutenett. Denne strukturen lages i CSS og ikke i HTML sammen med den andre kodingen til siden. Dette hjelper oss til å lage utforminger som kan omdefineres ved hjelp av medieforespørsler og tilpasse seg ulike sammenhenger. Grid lar oss skulle ordrenes elementer i kilden riktig fra sin visuelle presentasjon. CSS Grid kan brukes i alle de moderne nettlesere. Du kan se ett godt eksempel av en Grid Layout på det første bilde.

CSS Float CSS Float er en mer uforutsigbar og upresis layout. Når man floater elementet så legger det seg enten til høyre, venstre eller midt i. Det som er ulempen med Float kontra Flex og Grid er at de sistnevnte leggser seg akkurat i raden eller kolonnen du vil ha den, mens Float flyter fritt rundt til høyre, venstre eller i midten av innholdet.

CSS Flexbox Flexbox er en annen form for CSS layouts. Denne utformingen blir også brukt for å kunne få en god struktur på siden din, og det vil være lettere å plassere elementer. En av de store forskjellene fra Grid og Flexbox er at i Grid kan man bruke både kolonner og rader samtidig. Dette er ikke mulig i Flexbox. I Flex må man enten bruke kolonner eller rader. Dette gjør at du har litt mindre kontroll over hvor elementene legger seg, kontra om du bruker Grid. CSS Flexbox støttes også av alle de moderne nettleserne.

Table design var det første layout designet som kom. Der brukte man mer en tabell løsning for og får struktur på innholdet sitt. Det er veldig uoversiktlig, du kan ikke endre utseendet uten å endre HTML filen din og du er veldig låst til enkelte løsninger. Dette ble byttet ut med Float.

Å bruker slike CSS layouts gjør det lettere å få en fin struktur og det blir mye enklere å plassere elementene der du har lyst til å ha dem.
